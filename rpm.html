systemprompt.io — DNF/YUM repository

RPM repository for the systemprompt AI governance gateway. RHEL 9, Rocky 9, Fedora 40+, signed for x86_64 and aarch64.

Website Documentation Main repo Install guide Discord

Install

sudo curl -fsSL -o /etc/yum.repos.d/systemprompt.repo \
  https://rpm.systemprompt.io/systemprompt.repo
sudo rpm --import https://rpm.systemprompt.io/gpg.key
sudo dnf install systemprompt
Source-available under MIT for the template, BSL-1.1 for the core dependency. Compiled binary declares MIT AND BUSL-1.1.
