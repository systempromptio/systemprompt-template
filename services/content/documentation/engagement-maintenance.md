# Phase 2: Maintenance & Support

**Monthly Retainer — Begins at Production Deployment**

---

## Pricing

| | List Price | Partner Price |
|---|----------:|-------------:|
| **Monthly rate** | €2,000/month | **€1,200/month** |
| **Annual total** | €24,000 | **€14,400** |
| **Saving** | | **€9,600 (40%)** |

> **Reference implementation partner rate.** The 40% discount reflects Foodles's role as a flagship deployment partner.

**Payment:** €1,200/month, paid monthly in advance beginning at production deployment (Phase 2).

---

## What Is Covered

The maintenance retainer keeps the platform current, secure, and optimised throughout the engagement:

| Service | Description |
|---------|-------------|
| **Platform updates** | MCP specification changes, new model provider support, core platform improvements |
| **Security patches** | Vulnerability remediation, dependency updates, security advisory response |
| **Skill improvements** | Incremental improvements to existing skills and agent configurations |
| **Stakeholder reviews** | Monthly reviews covering usage, adoption trends, cost analysis, and action items |
| **Priority support** | Defined SLA for issue response and resolution |
| **Usage monitoring** | Proactive monitoring of usage patterns and cost optimisation recommendations |
| **AI landscape advisory** | Guidance on new models, protocol changes, and emerging best practices |

---

## What Is Not Covered

| Item | Notes |
|------|-------|
| **Major new feature development** | Scoped and quoted separately as additional consulting |
| **Infrastructure management** | Hosting, networking, and server management are Foodles's responsibility |
| **AI provider API costs** | API fees are billed directly by the provider (Anthropic, OpenAI, etc.) |
| **Third-party software licenses** | Any third-party tools required are Foodles's responsibility |

Major feature requests are welcome — they are scoped, quoted, and delivered as separate consulting engagements alongside the retainer.

---

## Cancellation

The maintenance retainer can be cancelled with **30 days' notice**.

Upon cancellation:

- The **platform license** remains active for the remainder of the 12-month term
- The **Control Center** extension code remains Foodles's property
- All skills, agents, configurations, and data remain with Foodles
- The platform continues to function — the retainer covers updates and support, not operational capability
