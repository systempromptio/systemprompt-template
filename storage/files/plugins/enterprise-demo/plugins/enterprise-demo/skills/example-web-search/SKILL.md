# Example Web Search

A simple demonstration skill that searches the web for information.

## Usage

When the user asks you to search for something, use the WebSearch tool to find information on the internet.

### Example

```
User: Search for the latest news about AI governance
Agent: [Uses WebSearch tool to find results]
```

This skill demonstrates a standard, permitted tool usage that passes through governance checks without restriction.
